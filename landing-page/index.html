如何用 Python 快速合并文件夹里的多个 Excel 表格？

可以使用 pandas 库。这是最简洁的写法：

import pandas as pd
import glob

# 读取所有xlsx文件
files = glob.glob("*.xlsx")
df = pd.concat((pd.read_excel(f) for f in files))
